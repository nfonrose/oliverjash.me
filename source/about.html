About

My name is Oliver, and I’m from a small town towards the south of England called Flitwick. Ever since I was about 13-years-old, I have lived and breathed the web.

Now at 20-years-old, I reside in London, England, where I make JavaScript web applications at @guardian. In my spare time, I'm also hard at work on my first startup.

I contribute code to GitHub. The code for this site is up on there, too, if you're interested. I post snippets of my thoughts on Twitter. Away from the computer, I take an interest in travelling and some photography. You can contact me at {{ site.email }} or via Twitter.

I really believe that computers can improve lives. I bear witness to it every day. But I believe we need to make technology more human. As a web developer, I want to be a part of that revolution.
